1.Обьясните своими словами, как вы понимаете, как работает прототипное
наследование в Javascript

В JS объекты имеют специальное скрытое свойство [[Prototype]], которое либо
равно null, либо ссылается на другой объект, этот объект называется прототипом.
При этом из объекта-прототипа новым объектом наследуются все свойства.
При вызове метода или попытке прочитать свойство obj, которого у него нет,
JS будет пытаться найти эти данные у прототипа этого obj.

Задавать свойства [[Prototype]] можно с помощью obj.__proto__ .

При вызове obj.method(), который наследуется из прототипа, this всё равно ссылается на obj.
Методы всегда работают с текущим объектом, даже если они наследуются.
С помощью цикла for..in можно перебрать свои и унаследованные свойства, остальные 
методы получения ключей/значений работают только с собственными свойствами объекта.
