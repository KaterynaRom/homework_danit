1. Описать своими словами для чего вообще нужны функции в программировании.

Функция содержит определенную последовательность команд (прописывается в теле функции), которая может вызываться 
неограниченное количество раз в том месте, где необходимо выполнить эти команды, без дублирования полного кода.
Частоиспользуемые способы объявления функции:
    - function declaration - можно вызывать в любом месте кода, даже до ее объявления.
        function имяФункции(параметры функции) {тело функции}
    - function expression - не имеет имени, нельзя вызывать до ее объявления, т.к. она объявляется через переменную.
        const имяПеременной = function(параметры функции) {тело функции}.

Функция может принимать значения - аргументы, и возвращать результат.

2. Описать своими словами, зачем в функцию передавать аргумент.

При вызове функции пользователь может передать ей данные, например, с помощью prompt. Функция принимает эти значения (они
же становятся аргументами ф-ции), обрабатывает их и после чего возвращает результат(значение) функции.
Таким образом, функция может обрабатывать за одним алгоритмом различные аргументы, которые можно изменять при каждом
последующем вызове функции.

Чтобы вернуть значение используют return, эти данные могут использоваться в дальнейшей работе кода.
Функция может содержать неопределенное количество параметров, значение которых может передаваться различными способами.
