1. Описать своими словами в несколько строчек, зачем в программировании нужны циклы.

Цикли позволяют выполнять одну и ту же часть кода несколько раз без необходимости дублировать его в коде. Работой цикла можно управлять с помощью операторов break(позволяет завершить выполнение цикла) и continue( позволяет перейти на следующую итерацию цикла).

В зависимости от поставленной задачи используют различные циклы:

-for - применяется, когда количество опереций известное.

-while - применяется, когда неизвестно количесто операций, цикл будет выполняться до тех пор пока условие истенное.

-do...while - аналогично циклу while, отличаеться от остальних тем, что этот цикли выполняется минимум один раз, даже если условие цикла не является истинным.
