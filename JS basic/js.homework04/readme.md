1. Опишите своими словами, что такое метод объекта.

Метод объекта - это свойство, значением которого являются встроенные в объект функции. 
Эти функции нужны для обработки тех данных, что существуют внутри объекта.

Объект = {
    свойство1: значение1,
    свойство2: значение2,
    свойствоn: значениеn,
    методОбъекта(параметры){
        тело функции
    }
}

Если нам нужно обратиться к свойству объекта внутри его метода, тогда перед именем
свойства нужно написать this. :
        this.имяСвойства
Если имя свойства состоит больше, чем с одного слова, тогда используется запись:
        this[имя свойства]

this указывает функции, что эта переменная находится среди свойств объекта, куда входит
эта функция.

Например, можно написать метод объекта, который будет брать данные о пользователе с 
объекта, рассчитывать возраст пользователя исходя из даты рождения, и выводить приветствие
по имени и фамилии, если пользователь старше 18 лет. 
Если в методе необходимо применить подход перебора всех свойств объекта, то используется 
цикл for...in, при этом тело цикла выполняется для каждого свойства объекта: 
        for (имяСвойства in имяОбъекта) {тело цикла}.
