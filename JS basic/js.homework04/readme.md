1. Опишите своими словами, что такое метод объекта.

Объект {...} - это тип данных, который содержит различную информацию в виде "свойство: значение", а также 
может содержать функции для обработки этих значений. Можно хранить большое количество свойств в одной переменной.
Данные внутри объекта можно изменять и добавлять. Свойства перечисляются через запятую. Ключи свойств должны быть
строками или символами, значения могут быть любого типа.
Свойства с целочисленными ключами при выведении сортируются по возрастанию, остальные располагаются в порядке
создания.

Чтобы обратиться к свойству объекта, указываем имя объекта и конкретное свойство через точку: "имя.свойство".
Чтобы использовать несколько слов для имени свойства объекта, его необходимо заключить в кавычки, вызываются же 
они таким образом: имяОбъекта["имя свойства"]. Если имя свойства является результатом выражения, то к нему
также можно обратиться лишь с использованием []: имяОбъекта[свойство]. Также [] используются для создания 
вычисляемого свойства.

Для удаления свойства объекта используют оператор delete.
Можно проверять существование свойства в объекте с помощью in: "имяСвойства" in имяОбъекта, результатом будет true,
если это свойство есть, либо false.

Для перебора всех свойств объекта используется цикл for..in, при этом тело цикла выполняется для каждого свойства
объекта:
for (имяСвойства in имяОбъекта) {тело цикла}
