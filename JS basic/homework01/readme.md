1.Обьяснить своими словами разницу между обьявлением переменных через var, let и const.
Область видимости переменной let только внутри блока {}, в котором она обьявлена; ее значения можно изменять, но нельзя обьявлять повторно в одном и том же блоке.
Если использовать let в цикле, то для каждой итерации будет создана своя переменная let, в отличии от var, когда на все итерации цикла существует одна переменная.
Через const задается константа, значение которой нельзя изменять, менять можно только свойства внутри ее; во всем остальном ведет себя как и let.
Переменная var существует и до своего объявления, ее значение в таком случае undefined. Переменную var можно обьявлять повторно.

2.Почему объявлять переменную через var считается плохим тоном?
При использовании var в разных областях видимости ее значение будет каждый раз переприсваиваться в отличии от let, когда каждая переменная существует только в своей области видимости.
Использование let позволяет избежать ошибок при использовнии одинаковых переменных в разных обласях видимости одного кода.
