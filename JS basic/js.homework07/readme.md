1. Опишите своими словами, как Вы понимаете, что такое Document Object Model (DOM)

Document Object Model - объектная модель документа, где все содержимое страницы
отображается в виде объектов. DOM представляет документ в виде структурированной 
группы узлов и объектов, у которых есть различные свойства и методы.

Каждый тег HTML-документа является узлом-элементом и формирует структуру документа, а любой
текст внутри тега - текстовым узлом.

С помощью объекта document - корневой объект документа, можно осуществлять множество 
операций(создавать, изменять, удалять и т.д.) над остальными объектами и узлами, используя 
специальные методы.
