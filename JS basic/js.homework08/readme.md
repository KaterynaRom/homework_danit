Обработчик событий - это функция, которая будет запускаться для элемента(элементов),
которому был назначен этот обработчик, при выполнении определенного события, 
например: click, mousedown(-up), mouseover(-out), mousemove, focus, keydown(-up) и другие.

Существуют три основных способа добавления обработчиков.
1. Через атрибут HTML: onclick="..."
2. Через DOM-свойство: elem.onclick = function
3. Через специальные методы: addEventListener/removeEventListener.
Преимуществом последнего способа является возможность назначать сразу несколько обработчиков
на одно событие, синтаксис:
   element.addEventListener(event, handler[, options]);
   где event-имя события, handler - ссылка на функцию-обработчик, options - дополнительный 
   (необязательный) объект с определенными свойствами.
removeEventListener работает аналогичным способом, но предназначен для удаления обработчика.
